# PDFEditor

A modern, fast, and feature-rich PDF editor built with Python, PyMuPDF, and CustomTkinter.

## Features
- **Multi-Tab Interface:** Edit multiple PDFs simultaneously.
- **Smart Splitting:** Split PDFs by exact Y-coordinates, percentages, or dynamically using OCR (Optical Character Recognition).
- **Batch Processing:** Apply operations (rotate, delete, crop, extract, split) to multiple pages at once.
- **Security:** Encrypt and decrypt PDF files.
- **Watermarks & Page Numbers:** Easily add text, images, watermarks, and page numbers.
- **Multi-Language Support:** Fully bilingual interface (English and Vietnamese).

## Installation
Run the included setup script to install dependencies:
```cmd
setup.bat
```
This will create a virtual environment and install all required packages.

## Usage
Run the application using the included runner script:
```cmd
run.bat
```
